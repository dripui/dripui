### Getting Started

> [!WARNING]
> Make sure your app is waterproof (or at least, water-resistent) before installing.

##### npm

```sh
npm add dripui
```

##### Yarn

```sh
yarn add dripui
```

##### pnpm

```sh
pnpm add dripui
```

##### Bun

```sh
bun add dripui
```
