### Getting Started

> [!WARNING]
> Make sure your app is waterproof (or at least, water-resistent) before installing.


##### npm
```sh
npm add dropui
```

##### Yarn
```sh
yarn add dropui
```
##### pnpm
```sh
pnpm add dropui
```

##### Bun
```sh
bun add dropui
```
