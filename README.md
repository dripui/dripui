## Quickstart

> [!WARNING]
> Make sure your app is waterproof (or at least, water-resistent) before installing.

##### npm

```sh
npm add @dripui/react
```

##### Yarn

```sh
yarn add @dripui/react
```

##### pnpm

```sh
pnpm add @dripui/react
```

##### Bun

```sh
bun add @dripui/react
```
